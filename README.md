https://ksessha.github.io/Kinosite/#

Стек технологий: 
  HTML5 – основная разметка страницы, структура интерфейса и модальных окон.
  CSS3 – стилизация элементов, адаптивная верстка для разных экранов, эффекты для модальных окон, кнопок и таймлайна. Используются Flexbox и Grid Layout.
  JavaScript (ES6+) – динамическое управление проектом:
    - добавление и удаление залов и фильмов,
    - работа с модальными окнами,
    - управление табами и сессиями,
    - drag & drop для сеансов.
